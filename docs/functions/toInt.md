# Function: toInt()

> **toInt**(`value`): `number`

Convert a string to a number, but less strict than Number() or parseInt().
Will remove all non-numeric characters, and also round the number if it has decimals.

## Parameters

• **value**: `string` \| `number`

## Returns

`number`
